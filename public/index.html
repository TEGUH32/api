Getting Started

To start using our API, simply make HTTP requests to the endpoints. All responses are in JSON format.

// Example: Testing server status
GET /api/status

// Example: Using Deepseek AI
GET /api/deepseek?q=Your question here

Response Format

All API responses follow a consistent format:

{ "status": true, "status_code": 200, "message": "Success message", "data": { ... }, "timestamp": "2024-01-15T10:30:00Z" }

Rate Limiting

API rate limits are set at 2000 requests per minute per IP address. If you exceed this limit, you'll receive a 429 status code.
